# Lumberjack Theme

Lumberjack Theme is a dark, high-contrast color theme for VS Code and Cursor.

## Preview

![Lumberjack Theme Logo](./lumberjack.png)

## Install

### VS Code Marketplace
1. Open Extensions.
2. Search for `Lumberjack Theme`.
3. Click Install.

### Cursor
1. Open Extensions.
2. Search for `Lumberjack Theme`.
3. Click Install.

## Activate
1. Open Command Palette (`Cmd/Ctrl+Shift+P`).
2. Run `Preferences: Color Theme`.
3. Select `Lumberjack Theme`.
